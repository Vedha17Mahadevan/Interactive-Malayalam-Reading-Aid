# 🧠 Malayalam Dyslexia Reading Assistant

An assistive reading system designed to support Malayalam readers, especially individuals with dyslexia, using interactive text, audio feedback, and speech-based validation.

---

## 🚀 Features

### 🔊 Text-to-Speech (TTS)
- Click any word to hear its pronunciation
- Play full sentence audio
- Uses Google Text-to-Speech (gTTS)

### 🖱️ Interactive Reading Panel
- Malayalam text is displayed as clickable words
- Words highlight when clicked
- Controlled UI (no typing allowed)

### 📊 Reading Difficulty Analysis
- Tracks how many times each word is clicked
- Identifies most difficult words
- Displays reading level (Easy / Moderate / Difficult)

### 🎤 Speech-to-Text (STT)
- User reads the text aloud
- System transcribes speech using SpeechRecognition
- Malayalam language support (`ml-IN`)

### ❌ Error Highlighting
- Compares spoken text with original text
- Incorrect words are highlighted in red
- Handles small Malayalam script variations

---

## 🧠 How It Works
```bash
Text Input → Word Interaction → Audio Feedback
↓
Speech Input
↓
Text Comparison
↓
Error Highlighting + Difficulty Score
```

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- gTTS (Text-to-Speech)
- SpeechRecognition (Speech-to-Text)
- PyAudio (Microphone input)
- Pygame (Audio playback)

---

## ⚠️ Limitations

- Malayalam speech recognition may not be fully accurate
- Minor mismatches may occur due to phonetic variations
- This system is assistive, not a medical diagnostic tool

---

## 🚀 Future Improvements

- Offline speech recognition (Vosk)
- Improved Malayalam normalization
- Phonetic-level pronunciation analysis
- Personalized learning feedback

---

## 📌 Usage

1. Select preset or enter Malayalam text  
2. Click words to hear pronunciation  
3. Use mic to read aloud  
4. View highlighted errors and reading analysis
